## Leadflow Agent

Leadflow Agent is an autonomous lead generation and automation planner for agencies and service businesses. Configure your ideal clients, offers, and automation priorities to generate high-fit lead lists, outreach sequences, and workflow recipes in seconds.

### Features

- Interactive planner that tailors lead scoring to your targets and budget stage
- Automation playbooks covering outbound, nurture, partner enablement, onboarding, and reporting
- Outreach cadence builder with channel recommendations and ready-to-send messaging hooks
- Recommended tech stack and next steps for immediate deployment

### Getting Started

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to explore the agent interface. Adjust the configuration panel to instantly regenerate lead plans.

### Production Build

```bash
npm run build
npm run start
```

The app is optimized for deployment on Vercel.
